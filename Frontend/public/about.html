Project Management System


In project management system the admin add students by uploading an excel file that contains the details of students which add them in the database the students can then login using the credentails mailed to them and then submit proposals for their project. A proposal is first approved by the admin and later by hod after this the admin has to assign guides to each group of students. Guides can then schedule presentation for the groups which are assigned to them and on completion of presentation they can alot marks to them .
